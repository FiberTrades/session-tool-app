Want to back up your data?

Your trades are saved in this browser's local storage. If you clear your browser data, switch computers, or your hard drive fails, that data is gone. A backup is a JSON file you can save to Google Drive, Dropbox, or anywhere else for safekeeping. You can re-import it later from the Import Data button at the bottom of the page.
Before enabling Auto-backup: backups go to your browser's default Downloads folder unless you change it. Set that to a Google Drive / Dropbox synced folder first so your backups land somewhere safe. Chrome / Edge users will be asked to pick a specific file instead.
